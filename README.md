## 🧑‍💻 About Me

Hi! I’m **Daniil Zhelanov**, a frontend developer focused on building clean, responsive and user-friendly web interfaces.

I completed my Bachelor’s degree at **Moscow Aviation Institute**, Faculty of **Computer Science and Applied Mathematics**, in **Applied Mathematics and Computer Science** from **2021 to 2025**.

Currently, I’m studying for a Master’s degree at **ITMO University**, Faculty of Software Engineering and Computer Systems, in the program **Assistive IT Solutions Development**.

I’m interested in modern frontend development, interface architecture, accessibility, design systems and creating web products that are both functional and visually polished.
